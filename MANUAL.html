Calculator Manual

This manual provides instructions on how to use the calculator.

Basic Arithmetic Operations

To perform basic arithmetic operations, use the number buttons (0-9) along with the following operators:

  • Addition: +
  • Subtraction: -
  • Multiplication: *
  • Division: /

Trigonometric Functions

The calculator supports trigonometric functions. To use them:

  • Press the trigonometric function button (sin, cos, tan).
  • Enter the angle in degrees.
  • Press the = button to calculate the result.

Clearing and Calculating

To clear the display, press the C button. To calculate the result, press the = button.
